Simple Program For Sorting The File System
Program Accepts Only 1 Optional Argument And That Should Be Path Of XML Configuration File For This Program. Otherwise, It Will Force On A File Named default.xmlgit
Let's Just Begin
XML Tags
There Are 4 Tags Has Defined By Default In SysSort:
< main-conf > : The Root Tag Of All Configurations
< config > : Configurations For The Working Directory
< dir > : Configurations For The Target Directory
< ignore > : Ignoring A Directory Or File From Any Actions
Now Let's Just Talk About Attributes
< config working-directory="" > : Required For Setting The Working Directory Of Configuration
< config delete-empty-directories="true|false" > : Optional Attribute For Deleting The Empty Directories, It's A Boolean Attribute (Default Is False).
< config hidden-files="true|false" > : Optional Attribute For Including Hidden Files In Actions (Default Is False).
< dir|ignore path="" > : < dir > Or < ignore > Both Will Use path For Their Action ( Which Has Defined )
Now Let's Talk About < dir >
< dir > < / dir > : Accepts A Body That Contains File Extensions To Move From Working Directory Specified In < config >

< dir path="/home/me/Pictures" > .gif|.jpg|.png|.svg < / dir >

Each Extension Specified Will Separate With A | Then All Files In Working Directory With One Of These Extensions Will Move Here.

< dir path="/home/me/Pictures/France" > -france.gif|-france.jpg|-france.png|-france.svg < / dir >

You Can Also Specify A Name Or Whatever You Want Before The Extension, Then It Will Move All Files Contains -france.svg.
